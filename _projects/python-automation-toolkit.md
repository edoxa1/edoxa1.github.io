## Overview

Aiogram-based bot for scheduling, made specifically for NU students. The bot can create and manage schedules conveniently through Telegram, allowing students to easily keep track of their classes, assignments, and other academic activities. It features an advanced Python backend with Redis and PostgreSQL, as well as highly optimized pdf parsing system for efficient data management and retrieval.

## Build Notes

- Dockerized backend for easy deployment and scalability.
- Redis and PostgreSQL for efficient data storage and retrieval.

## Обзор

Бот на aiogram для составления расписания, сделанный специально для студентов NU. Бот может создавать и управлять расписаниями удобно через Telegram, позволяя студентам легко отслеживать свои занятия, задания и другие учебные мероприятия. Он оснащен продвинутой Python-бэкенд-системой с Redis и PostgreSQL, а также высоко оптимизированной системой парсинга pdf для эффективного управления данными и их извлечения.

## Заметки по сборке

- Docker-бэкенд для легкого развертывания и масштабируемости.
